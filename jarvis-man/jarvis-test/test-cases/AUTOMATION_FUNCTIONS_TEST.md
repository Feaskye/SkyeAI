## 1. 测试用例：代码生成

### 1.1 测试目标

验证贾维斯系统的代码生成能力，确保系统能够根据用户需求生成各种编程语言的代码。

### 1.2 测试环境

- **硬件环境**：计算机设备
- **软件环境**：Java 17+, Go 1.20+, Ollama
- **网络环境**：有线网络或Wi-Fi

### 1.3 测试步骤

1. 启动贾维斯系统
2. 发送指令："帮我写一个Python脚本，计算1到100的和"
3. 发送指令："帮我写一个Java方法，实现斐波那契数列"
4. 发送指令："帮我写一个Go函数，计算两个数的最大公约数"
5. 发送指令："帮我写一个JavaScript函数，验证邮箱格式"
6. 检查生成的代码是否正确、完整、可运行

### 1.4 预期结果

- 系统能够正确理解用户的代码生成需求
- 系统能够生成语法正确的代码
- 系统能够生成功能完整的代码
- 系统能够生成符合最佳实践的代码
- 系统能够解释生成的代码功能
