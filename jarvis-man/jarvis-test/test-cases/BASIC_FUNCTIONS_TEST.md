# 基础功能测试用例

## 1. 测试用例：对话交互

### 1.1 测试目标

验证贾维斯系统的基本对话交互能力，包括文本输入和响应，确保系统能够理解用户的基本查询并提供合理的回复。

### 1.2 测试环境

- **硬件环境**：计算机设备
- **软件环境**：Java 17+, Go 1.20+, Ollama
- **网络环境**：有线网络或Wi-Fi

### 1.3 测试步骤

1. 启动贾维斯系统
2. 向系统发送文本查询："你好"
3. 向系统发送文本查询："你叫什么名字"
4. 向系统发送文本查询："你能做什么"
5. 向系统发送文本查询："今天天气怎么样"
6. 向系统发送文本查询："谢谢你"

### 1.4 预期结果

- 系统能够正确理解用户的基本查询
- 系统能够提供合理、自然的回复
- 系统响应速度适中（<2秒）
- 系统回复符合上下文语境

验证贾维斯系统的语音唤醒功能，确保系统能够通过语音指令被唤醒并响应。

### 2.2 测试环境

- **硬件环境**：计算机设备，麦克风
- **软件环境**：Java 17+, Go 1.20+, Whisper.cpp
- **网络环境**：有线网络或Wi-Fi

### 2.3 测试步骤

1. 启动贾维斯系统
2. 等待系统进入待机状态
3. 说出唤醒词："嘿贾维斯"

5. 唤醒后，说出指令："今天星期几"

- 系统能够通过唤醒词被成功唤醒
