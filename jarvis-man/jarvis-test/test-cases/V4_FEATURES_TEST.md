## 1. 测试用例：本地优先设计

### 1.1 测试目标

验证贾维斯系统的本地优先设计，确保系统能够在本地运行核心功能，保护用户隐私。

### 1.2 测试环境

- **硬件环境**：计算机设备
- **软件环境**：Java 17+, Go 1.20+, Ollama
- **网络环境**：有线网络或Wi-Fi，离线环境

### 1.3 测试步骤

1. 启动贾维斯系统
2. 发送指令："帮我执行一个本地计算任务"
3. 断开网络连接，进入离线模式
4. 发送指令："帮我计算1到1000的和"
5. 发送指令："帮我创建一个本地日程"
6. 重新连接网络，检查数据同步情况

### 1.4 预期结果

- 系统能够在离线模式下运行核心功能
- 系统能够在本地存储敏感数据
- 系统能够在网络恢复后自动同步数据
- 系统响应速度在本地模式下更快
- 系统能够保护用户隐私，敏感数据不离开本地设备
